The Paul Simpson API

Download this project as a .zip file Download this project as a tar.gz file

An API for accessing information for my application as a Junior Rails Developer at Harvest.

Introduction

I love APIs. I love them both for their systematic representation our data and for what they can become when used or combined in new and creative ways.

I also love Harvest. I have been watching them for months as I prepare to move to New York in June. The care they put into crafting their product, the open-minded and flexible approach they have to managing a distributed company, and the unwavering and vocal love of their users have made me certain that Harvest is the sort of place I want to work, learn, and grow as a developer.

I built this API for Harvest in an attempt to display myself in a way that makes me happy and would show the fine folks at Harvest how serious I am about working with them. Enjoy!

Supported Data Formats

The Paul Simpson API supports both JSON and XML requests. The data-type to be returned can be specified in two ways: either by setting the Accept and Content-type headers in the request (application/json for JSON, application/xml for XML) or by adding the desired content-type to the end of a request (i.e. /users/prsimp.json). If no content-type is specified, JSON will be returned.

API Wrapper Libraries

Sadly, no wrapper libraries currently exist for The Paul Simpson API.

Contact Information

Questions, comments, concerns, outrage, requests for interviews, and/or offers of employment should be directed to prsimp@gmail.com.

Usage

The root URL for The Paul Simpson API is http://api.prsimp.com. All requests are sent using the GET method. Ruby-style symbols are used to represent variable information in an API call. For instance, :username should be replaced with the username of the desired user.

User Information

Get basic biographical information about a user

GET /users/:username/whois

Response
{
  "name":"Paul Simpson",
  "email":"prsimp@gmail.com",
  "username":"prsimp",
  "age":27,
  "location":"Manhattan, Kansas",
  "url":"http://api.prsimp.com/users/prsimp"
}
